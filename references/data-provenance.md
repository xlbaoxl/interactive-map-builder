# Data provenance and redistribution

Record for every input:

- original path or public URL;
- source organization and dataset title when known;
- retrieval date for remote data;
- license or terms URL;
- input SHA-256;
- original and rendered feature counts;
- CRS and geometry operations;
- fields removed from the public output.

Do not redistribute source data merely because it is viewable online. Replace private, commercial, sensitive, or license-unclear examples with synthetic fixtures. Remove local absolute paths, access tokens, device identifiers, and unnecessary attributes from public artifacts.

Generated HTML embeds geometry and selected attributes. Treat it as a data release even when the source file is not included.
